什么是神经网络？

神经网络是一种受生物神经系统启发的计算模型：

  • 神经元 (Neurons): 基本计算单元
  • 层 (Layers): 神经元的组织结构
  • 权重 (Weights): 连接的强度
  • 激活函数 (Activation Functions): 非线性变换

为什么使用Python？

  • 简洁易读: 语法清晰，适合快速原型开发
  • 丰富的库: NumPy, SciPy, Matplotlib等
  • 社区支持: 大量的AI/ML资源和教程
  • 性能优化: 可以调用C/C++库提升性能
  • 工业标准: TensorFlow, PyTorch等主流框架

神经网络的基本组件

  1. 输入层 (Input Layer): 接收原始数据
  2. 隐藏层 (Hidden Layers): 特征提取和变换
  3. 输出层 (Output Layer): 产生预测结果
  4. 损失函数 (Loss Function): 衡量预测误差
  5. 优化器 (Optimizer): 更新网络参数

前向传播 (Forward Propagation)

数据从输入到输出的流动过程：


# 简单的前向传播示例
def forward(X, W, b):
    """
    X: 输入数据
    W: 权重矩阵
    b: 偏置向量
    """
    Z = np.dot(X, W) + b  # 线性变换
    A = sigmoid(Z)         # 激活函数
    return A

反向传播 (Backward Propagation)

计算梯度并更新参数：

  • 使用链式法则计算梯度
  • 从输出层向输入层传播误差
  • 更新权重和偏置

# 梯度下降更新
